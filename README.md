# Weather Forecast Application

## Description

A Weather Forecast Application built using Python, Tkinter, Requests, and OpenWeatherMap API.

The application allows users to enter a city name and also detect the location(ip address of internet provider but not user) and view:

* Temperature
* Humidity
* Wind Speed
* Weather Condition
* City Name

## Technologies Used

* Python
* Tkinter
* Requests
* Pillow (PIL)
* OpenWeatherMap API

## Features

* User-friendly GUI
* Background Image
* Weather Information Display
* Error Handling for Invalid City Names

## Installation

Install required libraries:

pip install requests pillow

## API Setup

1. Create an account on OpenWeatherMap.
2. Generate an API Key.
3. Open `main.py`.
4. Replace:

API_KEY = "YOUR_API_KEY"

with your actual API key.

## Run Project

python main.py
